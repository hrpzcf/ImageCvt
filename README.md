## 软件说明

用于监视目录，自动把所监视的目录里新增的 webp 文件转换为 jpeg/png 文件。

项目的图像转换模块 `ImageCvt/Library/libimgcvt.dll` 来自 [libimgcvt](https://github.com/hrpzcf/libimgcvt)，提供 JPEG/PNG/WEBP 图像之间互转的功能，详情请点击前往查看。
